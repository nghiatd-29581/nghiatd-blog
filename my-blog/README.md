# My Blog - Website Blog Kỹ Thuật

## Mô tả
Website blog về kỹ thuật với cấu trúc sách/chương/bài học. Sử dụng Next.js, Tailwind CSS, dark mode.

## Cài đặt
1. `npm install`
2. Set env vars in `.env.local`: ADMIN_USER=admin, ADMIN_PASS=pass
3. `npm run dev`

## Deploy
Push to GitHub, import to Vercel, set env vars.

## Update Nội dung
- Edit `data/books.json` cho metadata.
- Add HTML files vào `public/content/books/...`.
